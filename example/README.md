# ArithmeticKit Example

A simple Flutter app demonstrating the usage of ArithmeticKit package.

## Features

- Toggle between Basic and Scientific calculator modes
- Interactive examples for quick testing
- Real-time calculation results
- Error handling demonstration

## Running the Example

```bash
cd packages/ArithmeticKit/example
flutter pub get
flutter run
```

## Usage Examples

### Basic Calculator Mode

Try these expressions:
- `2+3` → 5
- `10-5` → 5
- `3×4` → 12
- `15÷3` → 5
- `10%3` → 1
- `2+3×4` → 14 (order of operations)
- `(2+3)×4` → 20 (parentheses)

### Scientific Calculator Mode

Try these expressions:
- `sin0` → 0
- `cos0` → 1
- `log100` → 2
- `ln(e)` → 1
- `2^8` → 256
- `sqrt16` → 4
- `π` → 3.14159...
- `e^2` → 7.389...
